Handling Events và Conditional Rendering

Tran Duc Trung 06/08/2018

    Handling Events trong React

    - Handling Events dùng để xử lý sử kiện trong React, nó cũng tương tự như handling event trên DOM Element, tuy nhiên có 1 số điểm khác nhau về cách sử dụng. Ví dụ:

     import React, {Component} from 'react';
     class App extends Component {
       handleAlertText(){
         alert('Hello World');
       }
      render() {
       return(
         <button onClick={ this.handleAlertText }>Click me</button>
       )
     }
    export default App

    - Ở ví dụ trên khi ta nhấn vào button Click me thì sẽ hiện ra thông báo 'Hello World', ở đây có 1 sự kiện là handleAlertText và ta đã gán nó vào button thông qua onClick. Khác với DOM Element là truyền vào onClick 1 string thì ở React ta truyền vào 1 JSX

    - this trong handle event: Khi ta sử dụng this trong handle event thì mặc định React sẽ không hiểu this đấy là gì và sẽ báo lỗi Undefined. Để có thể sử dụng this thì ta có 2 cách:

    • + Gán ngữ cảnh cho this bằng constructor:
      •  import React, {Component} from 'react';
       class App extends Component {
         constructor(props){
           super(props)
           this.state = {
             content: "Hello World"
           }
           this.handleAlertText = this.handleAlertText.bind(this)
         }

         handleAlertText(){
           alert(this.state.content);
         }
       }

       render() {
         return(
           <button onClick={this.handleAlertText}>Click me</button>
         )
       }
       export default App
    • + Sử dụng Arrow function:
      •  import React, {Component} from 'react';
       class App extends Component {
         constructor(props){
           super(props)
           this.state = {
             content: "Hello World"
           }
         }

         handleAlertText(){
           alert(this.state.content);
         }

         render() {
           return(
             <button onClick={() => this.handleAlertText()}>Click me</button>
           )
         }
       export default App

    - Ngoài việc sử dụng arrow function cho this thì nó còn được dùng trong trường hợp sự kiện có truyền tham số. Ví dụ:

     import React, {Component} from 'react';
     class App extends Component {
       constructor(props){
         super(props)
         this.state = {
           content: "Hello World"
         }
       }

       handleAlertText(title){
         alert(title);
       }

       render() {
         return(
           <button onClick={() => this.handleAlertText(this.state.content)}>Click me</button>
         )
       }
     export default App

    Conditional Rendering

    - Conditional Rendering hoạt động tương tự với Conditional trong Javascript. Trong React thường sử dụng javascript operator như là if để tạo các phần tử đại diện cho trạng thái hiện tại. Ví dụ:

     import React, {Component} from 'react';
     class App extends Component {
       render(){
         if (this.props.show) {
           return(<p>Hello World</p>)
         }
         return(<p>Show is false :(</p>)
       }
     }
     export default App;

    // index.js
     ReactDOM.render(
       <App show={false} />
       , document.getElementById('root')
     );

    - Trong ví dụ trên ta có thể thấy Component App phụ thuộc vào giá trị của props show

    - Trong React có thể dùng các biến để lưu trữ giá trị. Điều này giúp ta Condition render một phần của component trong khi những phần còn lại không hề thay đổi. Ví dụ:

     import React, {Component} from 'react';
     class App extends Component {
       constructor(props){
         super(props)
         this.state = {
           show: false,
         }
         this.handleToggle = this.handleToggle.bind(this)
       }

       handleToggle(){
         this.setState({
           show: !this.state.show
         })
       }

       render() {
         let contentelm = null
         if (this.state.show) {
           contentelm = (<<p>Hello World</p>)
         } else {
           contentelm = (<p>Content hidden</p>)
         }
         return(
           <div>
             <button onClick={this.handleToggle}>Show/Hide</button>
             {contentelm}
           </div>
         )
       }
     }
     export default App;
