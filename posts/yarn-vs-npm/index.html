Yarn vs Npm

Tran Duc Trung 23/07/2018

    Npm là gì?

    - NPM viết tắt của Node Package Manager là 1 công cụ dùng để quản lý thư viện lập trình javascript cho Nodejs

    Để chạy được phần mềm NPM hệ thống của chúng ta cần phải cài đặt NodeJS vì phần mềm NPM được tích hợp sẵn trong bộ cài NodeJS

    Để cài đặt một thư viện nào đó, chỉ cần mở cửa sổ CMD và thực thi lệnh:

    npm install <package name>

    Ví dụ:

    npm install react --save

    Và ta có thể sử dụng react trong mã Javascript bằng cách import:

    import React from 'react';

    Yarn là gì?

    YARN là công cụ quản lý thư viện javascript mã nguồn mở tốc độ cao, tin cậy và là 1 phiên bản cải tiến so với NPM. Được hình thành bởi các kỹ sư Facebook, Google...

    Cài đặt thư viện:

    yarn add <package name>

    Yarn vs Npm:

    - Cách thức hoạt động:

    • + File Yarn.lock: Cả yarn lẫn npm đều lưu các phiên làm việc (dependencies) của 1 project vào trong file package.json. Người dùng sẽ define 1 range của version để npm cài đặt phiên bản mới nhất. Điều này dẫn đến trường hợp 1 file package.json cài đặt module trên 2 máy khác nhau lại cho ra 2 phiên bản khác nhau của cùng 1 module, như vậy sẽ rất dễ xảy ra bug. Trong khi đó Yarn giải quyết vấn đề này bằng cách lưu chính xác module đã được cài đặt vào file Yarn.lock. Các máy khác khi cài đặt module sẽ tự động dựa vào file Yarn.lock để cài đặt chính xác phiên bản của module đã cài đặt trên máy kia.
    • + Cài đặt song song: Mỗi khi yarn và npm cài đặt 1 package, các tool này thực hiện 1 chuỗi task, Với npm, những task này được chạy trên mỗi gói và theo thứ tự lần lượt, nghĩa là nó sẽ chờ cho mỗi gói được cài xong mới chuyển sang cài gói tiếp theo. Trong khi đó Yarn chạy các task này đồng thời trên các gói cùng lúc nên tiết kiệm thời gian.
    • + Output với npm thường chứa nhiều thông tin không thực sự cần thiết. Trong khi output của Yarn trên terminal gọn hơn, và nếu muốn hiển thị output chi tiết hơn, người dùng có thể dùng comment khác.

    - Độ ổn định và tin cậy: Yarn hiện tại có độ ổn định tốt hơn so với npm
