Tìm hiểu về New Context Api

Tran Duc Trung 13/08/2018

    - React Context API là 1 tính năng mới được cập nhật trong phiên bản React 16.3

    - React Context cung cấp cho chúng ta cơ chế định nghĩa các data store và truy xuất chúng khi cần. Chúng ta không cần phải truyền data thông qua props nữa. Với React Context chúng ta có thể định nghĩa và sử dụng một thứ giống như là "global state" của ứng dụng vậy.

    - Ví dụ:

    • + Khởi tạo một Context Provider thông qua việc sử dụng hàm:React.createContext
      •  const AppContext = React.createContext()
       class AppProvider extends Component {
         constructor(props){
           super(props)
           this.state = {
             items: [
               {
                 name: 'Ruby On Rails',
               }
             ]
           }
         }

         render() {
           return (
             <AppContext.Provider value={this.state}>
               {this.props.children}
             </AppContext.Provider>
           )
         }
       }
    • + AppContext sẽ được sử dụng để tạo ra một context provider component. Provider này sẽ chứa dữ liệu chúng ta cần trong state của nó và nó sẽ gói toàn bộ nội dung của component App:
      •  class App extends Component{
         render(){
           return (
             <AppProvider>
               <List />
             </AppProvider>
           )
         }
       }
    • + Sau khi đã setup xong provider, chúng ta có thể truy xuất data bằng cách sử dụng Context Consumer
      •  class List extends Component{
         render(){
           return (<Item />)
         }
       }

       class Item extends Component{
         render(){
           return(
             <AppContext.Consumer>
               {(context) =>
                 item = context.items
                 <tr>
                   <td>{item.name}</td>
                 </tr>
               }
             </AppContext.Consumer>
           )
         }
       }

    - Như vậy là chúng ta không còn cần phải truyền prop items xuống component List hay Item nữa. Tất cả dữ liệu đã được xử lý bởi cơ chế của React Context.
