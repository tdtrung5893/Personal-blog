ECMAScript2015 (ES6) và Arrow Function

Tran Duc Trung 25/07/2018

    ECMAScript2015 (ES6) là gì?

    - ES6 là chữ viết tắt của ECMAScript 6, là một tập hợp các kỹ thuật nâng cao của Javascript và là phiên bản mới nhất của chuẩn ECMAScript

    - Những điểm mới của ECMAScript2015

    • + Arrow function: Tạo hàm bằng cách sử dụng dấu mũi tên =>
    • + Block Scoped: Định nghĩa biến với từ khóa let, cách định nghĩa này thì biến chỉ tồn tại trong phạm vi khối của nó (Block Scope)
    • + Destructuring Assignments: Khởi tạo các biến từ một mảng bằng một dòng code.
    • + Default Parameters: Gán giá trị mặc định cho các tham số.
    • + Template String: Tạo template HTML
    • + Weak, Set: các kiểu dữ liệu phức tạp mới
    • + Rest Parameters: Khai báo 1 hàm với số lượng tham số không xác định
    • + Multi-line String: Đơn giản là sử dụng dấu backticks để khai báo 1 chuôi có nhiều dòng
    • + Classes: Có thể viết Class và kế thừa, giống như là OOP
    • + Module: Thêm khả năng import và export
    • + Babel: ES6 vẫn còn chưa được nhiều trình duyệt hỗ trợ cho nên cần phải sử dụng trình biên dịch như babel để dịch lại sang mã ES5 để có thể chạy trên browser

    Thế nào là Arrow Function?

    - Arrow Function (Fat Arrow): Là một cách định nghĩa function mới trong ES6 bằng cách sử dụng dấu mũi tên =>. Nó giúp tiết kiệm thời gian phát triển và đơn giản hóa phạm vi function

    - Cách thức sử dụng Arrow Function:

    - Cách thông thường:

       function hello(name) {
         console.log('My name is' + name);
       }
       hello('Trung')    // My name is Trung

    - Trong ES6:

       var hello = (name) => {
         console.log('My name is' + name);
       }
       hello('Trung')    // My name is Trung

    - Nếu câu lệnh trong function là 1 câu lệnh đơn thì ta có thể bỏ đi dấu {}.

     var hello = (name) => console.log('My name is' + name);

    - Nếu trường hợp chỉ có 1 tham số thì ta có thể bỏ đi dấu ()

     var hello = name => console.log('My name is' + name);

    - Trường hợp không có tham số thì bắt buộc phải sử dụng cặp ngoặc tròn ()

     var hello = ()    => console.log('Hello world');    // Error

    - Ràng buộc của mũi tên: Mũi tên => phải đứng ngay sau dấu đóng ngoặc tròn trong phần khai báo tham số, không được phép xuống dòng

     var hello = () => console.log('Hello world');

    - Những điều cần lưu ý:

    • + This: Arrow function tạo ra hành vi của thuộc tính this. Các phương thức call (), apply (), và bind () sẽ không thay đổi giá trị của this trong arrow function
    • + Constructors: Arrow function không thể được sử dụng làm constructor như function. Nếu khởi tạo new với arrow function, nó sẽ báo lỗi. Arrow function không có thuộc tính prototype hay internal methods
