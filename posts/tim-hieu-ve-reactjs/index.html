Tìm hiểu về ReactJs

Tran Duc Trung 03/08/2018

    ReactJs gì?

    - ReactJs là 1 thư viện JavaScript được tạo ra bởi facebook và instagram, nó cho phép các developer tạo ra giao diện người dùng (UI)

    - Các lí do nên sử dụng React:

    • + React tạo ra cho chính nó DOM ảo và ReactJS cũng tính toán những thay đổi nào cần cập nhật lên DOM và chỉ thực hiện chúng ~> giúp cải thiện hiệu suất rất nhiều
    • + React sử dụng cú pháp JSX giúp cho việc viết các đoạn code JS dễ dàng hơn. JSX là kết hợp của JavaScript và XML, tức là ta có thể thêm các đoạn HTML vào trong các hàm render mà không cần phải nối chuỗi.
    • + Có hỗ trợ việc debug code thông qua extension React Developer Tools trên trình duyệt Chrome
    • + ReactJS có thể vừa render ở ngoài trình duyệt sử dụng DOM và cũng có thể render bằng các chuỗi HTML mà server trả về
    • + Hiệu năng cao đối với các ứng dụng có dữ liệu thay đổi liên tục, dễ dàng cho bảo trì và sửa lỗi

    Cài đặt ReactJs

     npm install --save react react-dom react-router

    Để sử dụng React thì ta dùng lệnh import:

     import React from 'react';

    JSX là gì? Sử dụng JSX như thế nào?

    - JSX là 1 cú pháp mở rộng cho JavaScript, là kết hợp của JavaScript và XML. Nó chuyển đổi cú pháp dạng gần như XML về thành JavaScript, giúp người lập trình có thể code ReactJS bằng cú pháp của XML thay vì phải dùng JavaScript

    - Cú pháp của tài liệu XML được xây dựng dựa vào cấu trúc NODE lồng nhau, mỗi node sẽ có một thẻ mở và một thẻ đóng:

     <nodename> content </nodename>
    • + <nodename> là thẻ mở, tên của thẻ này do bạn tự định nghĩa.
    • + </nodename> là thẻ đóng, tên của thẻ này phải trùng với tên của thẻ mở.
    • + content là nội dung của thẻ này

    - Ta cũng có thể sử dụng cú pháp tự đóng thẻ giống như trong XML <nodename />

    - Ta có thể nhúng bất kỳ biểu thức JavaScript nào trong JSX bằng cách đóng nó trong dấu ngoặc nhọn {}

     const user = {
       firstName: 'Trung',
       lastName: 'Tran'
     };
     const element = (
       <h1>
         Hello, { user.firstName }!
       </h1>
     );

     ReactDOM.render(    // render element tới thẻ có Id là root
       element,
       document.getElementById('root')
     );

    - Sau khi biên dịch thì biểu thức JSX sẽ trở thành các đối tượng JavaScript. Ví dụ:

     function greeting() {
       return(
         <div>
           <button className="btn">Submit</button>
         </div>
       );
     }

    React Components là gì?

    - Components là 1 trong những thành phần quan trọng nhất của Reatjs, cho phép ta chia nhỏ đoạn code của UI thành những phần độc lập với nhau, với mục đích để tiện cho việc quản lý và tái sử dụng.

    - Cú pháp Component: Tên chữ cái đầu tiên của Component phải viết hoa

    • + Dùng let/const:
      •  let App = () => {
         return(
           <h1>Hello world</h1>
         );
       }
    • + Dùng function:
      •  function App2() {
         return(
           <h1>Hello world</h1>
         );
       }
    • + Dùng class:
      •  class App3 extends Component {
         render() {
           return(
             <h1>Hello world</h1>
           );
         }
       }

    - Cách sử dụng: Sau khi đã khởi tạo 1 Component thì tên của Component đó sẽ được dùng để làm tên 1 thẻ tag giống như là các thẻ tag trong html

    VD: ta có file index.html với đoạn code sau:

     <div id="root"><div>

    Trong file index.js ta sẽ khai báo Component và gọi nó ra trong hàm ReactDOM.render

     import React, { Component } from 'react';
     class App extends Component {
       render() {
         return(
           <h1>Hello world</h1>
         );
       }
     }

     ReactDOM.render(
       <App />
       , document.getElementById('root')
     );

    - Ngoài ra thì mỗi Component trong React đều 1 state riêng và các props có thể thay đổi được và React sẽ thực hiện cập nhật Component dựa trên những thay đổi của state.

    Props là gì?

    - Props là viết tắt của Properties, nó là thuộc tính của 1 Component, ta có thể truyền Props vào 1 component từ component cha của nó. Props có thể là 1 object, funtion, string, number.....

    - Trong React thì Props là bất biến, tức là nó không thể thay đổi được.

    VD: Ta có 1 file Content.js

     import React, {Component} from 'react';
     class Content extends Component {
       render(){
         return(
           <div>{this.props.title}</div>
         );
       }
     }
     export default Content;

    Và 1 file App.js

     import React, {Component} from 'react';
     import Content from "./Content";
     class App extends Component {
       render(){
         return(
           <Content title="This is Props Component Content" />
         );
       }
     }
     export default App;

    - Ngoài ra thì Component còn có 1 props đặc biệt nữa là children, khác với các thuộc tính bình thường thì children được truyền vào bằng cách đặt giữa 2 thẻ đóng và thẻ mở của 1 Component. Ví dụ:

     import React, {Component} from 'react';
     class Content extends Component {
       render(){
         return(
           <div>{this.props.children}</div>
         );
       }
     }
     export default Content;

     import React, {Component} from 'react';
     import Content from "./Content";
     class App extends Component {
       render(){
         return(
           <Content>This is Children of Component Content</Content>
         );
       }
     }
     export default App;
