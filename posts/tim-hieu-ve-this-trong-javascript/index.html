Tìm hiểu về this trong JavaScript

Tran Duc Trung 20/07/2018

    Phân biệt function expression với function statement?

    - Trong JavaScript con trỏ this dùng để tham chiếu đến đối tượng đang chứa mã lệnh thực thi.

       function person(age, name) {
         this.myage = age;
         this.myname = name;
       }
       var people = new person('25', 'Trung')
       alert(people.myage);    // 25
       alert(people.myname);    // Trung

    - Con trỏ this trong trường hợp gán hàm cho một biến khác:

       var name = "Duc";
       var person {
         name: "Trung",
         myname: function() {
           alert(this.name);
         }
       };
       var newperson = person.myname;
       newperson();    // Duc

    -> Khi ta gán hàm cho 1 biến khác thì sẽ xảy ra tình trạng thay đổi ngữ cảnh của hàm nếu được gọi. Để giữ nguyên ngữ cảnh thì ta sử dụng hàm bind:

       var name = "Duc";
       var person = {
         name: "Trung",
         myname: function() {
           alert(this.name);
         }
       };
       var newperson = person.myname.bind(person);
       newperson();    // Trung

    - Con trỏ this trong hàm mượn:

       var object = {
         name: "Duc",
         myname: function() {
           alert('My name is' + ' ' + this.name);
         }
       };

       var person = {
         name: 'Trung',
       }

       object.myname.apply(person);    // My name is Trung

    -> Khi muốn dùng hàm myname để hiển thị name của đối tượng person thì ta sẽ dùng hàm apply để thay đổi ngữ cảnh.

    - Con trỏ this trong sử dụng hàm callback:

       var person = {
         name: "Trung",
         myname: function() {
           return this.name;
         }
       };
       $(button).click(person.myname);    // Undefined

       var person = {
         name: "Trung",
         myname: function() {
           return this.name;
         }
       };
       $(button).click(person.myname.bind(person));    // Trung

    -> Ở đây, khi gọi hàm myname thì this sẽ bị Undefined do this không thuộc ngữ cảnh của person, do vậy, để có thể thực thi thì ta phải gán ngữ cảnh cho this băng hàm bind.
